# AI Email Automation Bot

An AI-powered email automation workflow built using n8n, Gmail API, and OpenAI GPT models.

## Features
- Detects incoming Gmail messages automatically
- Extracts email content
- Uses AI to classify emails
- Generates contextual AI replies
- Sends automatic responses to the original sender

## Tech Stack
- n8n
- OpenAI API
- Gmail API
- Workflow Automation

## Workflow
Gmail Trigger → OpenAI → Gmail Send

## Use Cases
- AI email assistant
- Smart auto-reply system
- Customer support automation
- AI workflow automation
